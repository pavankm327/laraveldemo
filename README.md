## Commands used

- **composer create-project --prefer-dist laravel/laravel:^7.0 laraveldemo**
- **php artisan make:controller UserController**
- **php artisan make:model Accounts --migration**
- **php artisan migrate**
- **php artisan db:seed**
- **php artisan serve**
